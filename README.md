## 写在最前面，极其重要：
<u> **当上传非公开的、不来源于自己的劳动成果时，务必争得他人同意！！** </u>  
+ 比如：**把考试卷、老师自己整理/出的作业题、老师做的作业答案传上来，要老师同意**；把老师发的教材传上来，要老师同意，万一有版权问题，等等。  
+ 上传此类资料，请邮箱、微信询问利益相关者。  
+ 但上传教材的zlibrary链接、自己写的论文、做pre的内容，你自己愿意就好。  
+ <u>各种情况不胜枚举，但若不确定，先别传，先去提问箱里问。</u>（但应该没啥灰色地带的？）
+ 存在侵权，请在项目的Issue模块指出。
